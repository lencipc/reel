# Reels Dashboard

Një aplikacion Next.js për menaxhimin e reels videosh.

## Instalimi

```bash
npm install
npm run dev
```

## Features

- 📹 Upload dhe menaxhim i videos
- 🎬 Dashboard për shikimin e të gjithë posts
- 🔗 Shikimi individual i postave
- 📱 Responsive design

## API Routes

- `POST /api/upload` - Upload i video postave
- `GET /api/posts` - Përshtjem i të gjithë postave
- `GET /api/og` - Open Graph image
